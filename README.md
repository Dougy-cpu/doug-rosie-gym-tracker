# Doug & Rosie Gym Tracker

Private mobile-first gym habit tracker for Doug and Rosie.

## Routes

- `/doug`
- `/rosie`
- `/couple`

## Local Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Set `DATABASE_URL` to a PostgreSQL database before using the tracker.

## Checks

```bash
npm test
npm run typecheck
npm run build
```
